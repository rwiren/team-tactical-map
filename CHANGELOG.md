# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-06-17

### Added
- Initial release — broadcast football to 2D tactical radar
- YOLOv8 player + ball detection
- ByteTrack persistent tracking IDs
- K-means jersey color clustering (team separation)
- Manual 4-point homography for pitch registration
- 2D radar mini-map rendering (matplotlib + OpenCV)
- Colab notebook with YouTube clip download
- Sample output from Finland vs. match broadcast
- Multi-sport project structure (football, basketball, hockey, military placeholders)
